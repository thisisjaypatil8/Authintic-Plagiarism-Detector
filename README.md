
# ğŸ§  Authintic: AI-Powered Plagiarism Detector

**Authintic** is a comprehensive, full-stack application designed to detect plagiarism in text documents.  
It uses a sophisticated, multi-layered approach that combines:

- A **Node.js backend** for authentication, file management, and database operations  
- A **Python/Flask NLP microservice** for semantic similarity detection  
- A **React frontend** for intuitive user interaction  

This system is built to identify both **direct copy-paste plagiarism** and **advanced paraphrasing** using **sentence-level semantic embeddings** and a **high-speed vector search index**.
